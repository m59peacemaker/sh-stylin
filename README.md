# stylin

Customizes the color and formatting of input from stdin.

## Install
```
sudo curl -o /usr/local/bin/stylin https://raw.githubusercontent.com/m59peacemaker/sh-stylin/${tag}/stylin && \
sudo chmod +x /usr/local/bin/stylin
```
